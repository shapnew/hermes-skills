# Hermes Agent Custom Skills

基于 [Hermes Agent](https://hermes-agent.nousresearch.com/) 框架构建的 AI Agent 自定义技能系统。

## 项目简介

本项目展示了我使用 Hermes Agent 进行 AI 驱动自动化运维和开发管理的实践经验。通过创建自定义技能（Skills），让 AI Agent 能够自主完成服务器监控、配置管理、故障排查等复杂任务。

## 实际应用场景

### FoxAPI AI API Gateway（已上线运营）

我搭建并运营的 AI API 网关平台，基于 sub2api + new-api 构建：
- 为开发者提供稳定的模型 API 代理服务
- 使用 Hermes Agent 进行日常运维管理
- 实现 AI 驱动的自动化配置和故障响应

### 核心技能

| 技能 | 说明 |
|------|------|
| **brainstorming** | 需求打磨与设计确认。在实施任何操作前先与用户充分沟通，确保方向正确 |
| **更多技能开发中** | 持续积累最佳实践 |

## 自动化运维实践

- CRON 定时巡检：服务器资源监控、异常告警
- 配置管理：sub2api / new-api 配置自动化
- 故障排查：AI 驱动的系统性根因分析
- 文档维护：VitePress 技术文档自动生成

## 技术栈

- **AI Agent框架**: Hermes Agent
- **API网关**: sub2api / new-api
- **服务器**: Ubuntu 24.04 / Nginx
- **模型**: DeepSeek / MiMo
- **集成平台**: QQ / WeChat
